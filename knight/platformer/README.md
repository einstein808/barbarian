Mini Platformer usando PixiJS

Como usar:

1. Abra `platformer/index.html` no navegador (duplo clique ou arraste para o navegador).
2. Controles: `A` / `D` ou `←` / `→` para mover; `Espaço` / `W` / `↑` para pular.

O projeto usa PixiJS via CDN para facilitar. Próximos passos sugeridos:
- Substituir o retângulo do jogador por um sprite (usar `knight/idle.png` ou outro).
- Adicionar inimigos, colher frutas e pontos.
- Ajustar física (gravidade, velocidade) conforme desejado.
